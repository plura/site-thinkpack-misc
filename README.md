# Thinkpack Misc — Placeholder Sites

Placeholder site shared by two domains:

- **EcoPaper** — [ecopaper.pt](https://ecopaper.pt/)
- **PlanetPack** — [planetpack.eco](https://planetpack.eco/)

## Deploy

Uses the [VS Code SFTP extension](https://marketplace.visualstudio.com/items?itemName=Natizyskunk.sftp). Copy `.vscode/sftp.json.example` to `.vscode/sftp.json`, fill in the credentials, then use **SFTP: Upload Project** with the appropriate profile.
